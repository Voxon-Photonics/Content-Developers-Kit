Adding these keys to your registry will enable an 'Open in VoxieOS' option when you right click. This feature can be for when you right click on Windows background or on/within a folder.

For the right click desktop registry file the key added is:

HKEY_CLASSES_ROOT\Directory\Background\shell\Voxon

For the right click within/on a folder the reg key added is:

HKEY_CLASSES_ROOT\Directory\shell\Voxon
