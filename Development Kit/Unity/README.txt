Its recomended that you use Unity version 2018 with the Voxon Unity Plugin.
To use the plugin simply follow these steps 

1) open or start a new Unity Project 
2) In the file menu bar click on 'Assets'
3) Navigate to 'Import Package' and select 'Custom Package...'
4) Use the file browser to find the Voxon Unity Package (default is under C:\Voxon\Development Kit\Unity)
5) Click the Import button
