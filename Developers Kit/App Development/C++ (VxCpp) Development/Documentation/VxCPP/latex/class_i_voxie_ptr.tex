\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_i_voxie_ptr}{IVoxie\+Ptr}} classes which inherit this interface require access to \mbox{\hyperlink{class_i_voxie_box}{IVoxie\+Box}} pointer.
