\doxysubsection{Detailed Description}
Voxie Window Struct -\/ Configuration for the Voxon Window. This struct manages all the settings to do with the volume. 

Most settings to do with creating a volume are contained here. This struct is passed in when \mbox{\hyperlink{class_voxie_box_a9df7e95b7c877b5942690affef94d4a6}{Voxie\+Box\+::init()}} is called. Most of these settings can be updated while the volume is rendering. Pass the \mbox{\hyperlink{structvoxie__wind__t}{voxie\+\_\+wind\+\_\+t}} struct through Voxie\+Box\+::update() for the settings to be applied to the current volume. \begin{DoxySeeAlso}{See also}
