To use the Between Worlds demo in a new Unity Project:

1) Create a new Unity project using Unity verision 2018.4 or greater  
2) Import the Voxon Unity Package 
3) Import the Between World Plugin
4) Open up the Between World Scene
5) Click on 'Voxon' menu from the top navgation bar and select 'Process'  
6) Within the Process manager settings make sure 'Editor Camera' is set to _camera (the Voxon camera for this scene)
